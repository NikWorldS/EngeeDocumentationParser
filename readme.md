# Engee Documentation Parser
Парсер необходим для скачивания документации блоков (страниц блоков) Engee в формате markdown.

Есть фильтр библиотек, блоки из которых будут скачиваться
(в базовой настройке **выключены** библиотеки _RITM_, _Оборудование_)

При скачивании файлов из текста убираются заголовки "примеры", "дополнительные возможности", "смотрите также".
